# Supply-Chain-Analysis-Dashboard
## Project Objective
An interactive Supply Chain Dashboard built using Microsoft Power BI to analyze orders, revenue, profit, inventory, supplier performance, and delivery efficiency, providing actionable insights for data-driven supply chain decision-making.

## Data Used
-<a href="https://github.com/skeethika/Supply-Chain-Analysis-Dashboard/blob/main/DataCoSupplyChainDataset.zip">dataset</a>

##Project Overview

The dashboard provides a comprehensive view of supply chain performance by tracking:

Total Orders
Revenue and Profit
Delivery Performance
Inventory Levels
Supplier Performance
Regional Sales Trends

##Key Insights

Identified top-performing products and regions.
Monitored on-time delivery rates and delayed shipments.
Analyzed profit margins across categories.
Evaluated supplier efficiency and reliability.
Detected trends in order volume and customer demand.

##Tools & Technologies Used

Microsoft Power BI
Microsoft Excel
Data Cleaning and Transformation with Power Query
DAX Measures and KPIs

## Dashboard Features

Interactive slicers and filters
KPI cards
Trend analysis charts
Geographic visualizations
Drill-through reports

## Business Impact

This dashboard helps businesses:

Optimize inventory management
Improve delivery efficiency
Reduce operational costs
Enhance supplier relationships
Increase profitability

Developed a professional dashboard that converts raw supply chain data into actionable insights for strategic decision-making.
