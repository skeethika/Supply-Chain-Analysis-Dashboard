# Supply-Chain-Analysis-Dashboard
## Project Objective
An interactive Supply Chain Dashboard built using Microsoft Power BI to analyze orders, revenue, profit, inventory, supplier performance, and delivery efficiency, providing actionable insights for data-driven supply chain decision-making.

## Data Used
-<a href="https://github.com/skeethika/Supply-Chain-Analysis-Dashboard/blob/main/DataCoSupplyChainDataset.zip">dataset</a>

##Project Overview

The dashboard provides a comprehensive view of supply chain performance by tracking:

Total Orders
Revenue and Profit
Delivery Performance
Inventory Levels
Supplier Performance
Regional Sales Trends
